# Emergency Operations and Patient Flow Dashboard
Designed and developed a three-page executive Power BI dashboard for the FP20 Analytics Challenge, using DAX, data modeling, and ZoomCharts to analyze emergency operations, workforce performance, and financial outcomes.

## Features

-Interactive dashboard navigation
-Dynamic KPI tracking
-Cross-filtering analytics
-Region-to-hospital drill-down
-Executive performance monitoring
-Operational trend analysis
-Hospital benchmarking
-Multi-page reporting experience

## Technologies

- Power BI
- DAX
- Power Query
- SQL
-ZoomCharts
